    6. HTMLMarqueeElement.behavior

    Sets how the text is scrolled within the marquee. Possible values are scroll, slide and alternate. If no value is specified, the default value is scroll. 

    <marquee behavior="alternate">Hey! I am a marquee text</marquee>
<marquee behavior="scroll">Hey! I am a marquee text</marquee>
<marquee behavior="slide">Hey! I am a marquee text</marquee>

    HTMLMarqueeElement.bgColor

    Sets the background color through color name or hexadecimal value.

    <marquee bgColor="red">Hey! I am a marquee text</marquee>

    HTMLMarqueeElement.direction

    Sets the direction of the scrolling within the marquee. Possible values are left, right, up and down. If no value is specified, the default value is left. 

    <marquee direction="down">Hey! I am a marquee text</marquee>
<marquee direction="up">Hey! I am a marquee text</marquee>
<marquee direction="right">Hey! I am a marquee text</marquee>
<marquee direction="up">Hey! I am a marquee text</marquee>

    HTMLMarqueeElement.height.width

    Sets the height and width in pixels or percentage value.

    <marquee height="100px" width="200px">Hey! I am a marquee text</marquee>

    HTMLMarqueeElement.hspace.vspace

    Sets the horizontal and vertical margin.

    <marquee vspace="50px" hspace="70px">Hey! I am a marquee text</marquee>

    HTMLMarqueeElement.scrollAmount

    Sets the amount of scrolling at each interval in pixels. The default value is 6.

    <marquee scrollAmount="100">Hey! I am a marquee text</marquee>

    HTMLMarqueeElement.scrollDelay.trueSpeed

    By default, scrollDelay values lower than 60 are ignored. If trueSpeed is true, then those values are not ignored. Sets the interval between each scroll movement in milliseconds. The default value is 85. Note that any value smaller than 60 is ignored and the value 60 is used instead, unless trueSpeed is true. 

    <marquee scrollDelay="90" trueSpeed="true">Hey! I am a marquee text</marquee>

    HTMLMarqueeElement.loop

    <marquee loop>Hey! I am a marquee text</marquee>

HTML

What is HTML?
HTML stands for Hyper Text Markup Language. It is the standard markup language for creating web pages and web applications.
What are the new features in HTML5?
HTML5 introduced several new features such as `<canvas>`, `<video>`, `<audio>`, `<article>`, `<header>`, `<footer>`, `<nav>`, `<section>`, etc. It also includes new form input types like `<date>`, `<email>`, `<url>`, etc.
What is the difference between `<div>` and `<span>`?
`<div>` is a block-level element used to group elements and create divisions in a web page layout. `<span>` is an inline element used to apply styles to a part of a text or a line.
What are semantic HTML tags? Can you give examples?
Semantic HTML tags are tags that clearly describe their meaning in the web page structure. Examples include `<header>`, `<nav>`, `<section>`, `<article>`, `<footer>`, `<aside>`, etc.
What is the purpose of the `alt` attribute in `<img>` tags?
The `alt` attribute provides alternative text for an image, which is displayed if the image fails to load. It also improves accessibility for users with screen readers.
Explain the difference between `<script>`, `<script async>`, and `<script defer>`?
`<script>` loads and executes the script synchronously, blocking the HTML parsing. `<script async>` loads the script asynchronously, allowing the HTML parsing to continue. `<script defer>` also loads the script asynchronously but ensures that the script is executed after the HTML is parsed.

CSS

What is CSS?
CSS stands for Cascading Style Sheets. It is used to style the layout and presentation of web pages.
What is the box model in CSS?
The CSS box model describes the design and layout of elements in a webpage. It consists of the content area, padding, border, and margin.
What is the difference between `margin` and `padding`?
`Margin` is the space outside the border of an element, while `padding` is the space inside the border of an element.
What is the difference between `inline`, `block`, and `inline-block` in CSS?
`inline` elements flow along with the text and do not start on a new line. `block` elements start on a new line and take up the full width available. `inline-block` elements flow along with the text but can have set widths and heights like block elements.
How do you center an element horizontally and vertically in CSS?
To center horizontally, you can use `margin: 0 auto;` for block-level elements with a defined width. To center vertically, you can use the Flexbox or Grid layout properties, such as `align-items: center;` for Flexbox.
What is the `box-shadow` property in CSS used for?
The `box-shadow` property adds a shadow effect to an element's box. It allows you to set the color, blur radius, spread radius, and position of the shadow.

JavaScript

What is JavaScript?
JavaScript is a programming language that enables interactive and dynamic content on web pages.
What are the different data types in JavaScript?
JavaScript has several data types including `string`, `number`, `boolean`, `null`, `undefined`, `object`, and `symbol`.
Explain the difference between `==` and `===` in JavaScript?
`==` is used for equality comparison, which converts the operands to the same type before making the comparison. `===` is strict equality comparison, which checks both the value and the type of the operands.
Explain what event bubbling and event capturing are.
Event bubbling is the process where an event starts at the most specific element and flows outward to the least specific element. Event capturing is the opposite; the event starts from the least specific element to the most specific.
What is a closure in JavaScript? Can you give an example?
A closure is a function that has access to its own scope, the outer function's scope, and the global scope. It "remembers" its lexical environment even when the function is executed outside that scope. Example: ```javascript function outerFunction() { let outerVar = "I am from the outer function"; function innerFunction() { console.log(outerVar); } return innerFunction; } let closureExample = outerFunction(); closureExample(); // Outputs: I am from the outer function ```
What is JSX?
JSX (JavaScript XML) is a syntax extension for JavaScript that allows HTML code to be written inside JavaScript code.

ReactJS

What is React?
React is a JavaScript library for building user interfaces, developed by Facebook. It allows developers to create reusable UI components.
What are the key features of React?
Some key features of React include virtual DOM for performance optimization, component-based architecture, one-way data flow, and reusable components.
What is the purpose of `useEffect` in React?
`useEffect` is a hook in React that allows you to perform side effects in function components. It is used to handle tasks such as data fetching, subscriptions, or manually changing the DOM.
What is the difference between functional components and class components in React?
Functional components are simpler and use only JavaScript functions. Class components are more feature-rich and can hold state, lifecycle methods, and more.
What are props in React?
Props (short for properties) are used to pass data from parent components to child components in React. They are read-only and help in making components reusable.
Explain the concept of state in React.
State in React is an object that represents the parts of the app's data that can change over time. It is managed within a component and can be updated using the `setState` method.

General Questions

Explain the difference between `GET` and `POST` requests.
`GET` requests are used to retrieve data from a server, while `POST` requests are used to send data to a server to create/update a resource.
What is responsive web design?
Responsive web design is an approach to design and development that ensures web pages render well on various devices and window or screen sizes.
What tools do you use for debugging and testing your code?
Tools like Chrome DevTools, React DevTools, Jest, Enzyme, and others are commonly used for debugging and testing frontend code.
What is the purpose of version control systems like Git?
Version control systems like Git help developers track changes to their code, collaborate with others, revert to previous versions, and maintain a history of changes.
Explain the difference between HTTP and HTTPS.
HTTP (Hypertext Transfer Protocol) is a protocol used for transmitting data over the internet. HTTPS (Hypertext Transfer Protocol Secure) is a secure version of HTTP that encrypts the data transmitted between the client and the server.
How do you optimize a website's performance?
Website performance can be optimized by minimizing HTTP requests, using browser caching, optimizing images, using a content delivery network (CDN), minifying and compressing files, and employing efficient code practices.
